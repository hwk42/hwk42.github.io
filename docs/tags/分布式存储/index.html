云原生存储-使用NFS动态提供PV

K8s有两种提供PV的方式：静态和动态。静态供应是指由管理员手动创建，并指定存储类的类型和容器，PVC根据要求进行绑定；动态供应则无需手动创建，在部署provisioner后即可监听PVC事件，自动完成PV创建、绑定、释放等生命周期，对运维和存储利用率提升很友好。K8s内置了多种存储支持动态供应，常用的包括Ceph、NFS等，本文记录下NFS动态供应的搭建过程，并对比两者的优势和不足作为选型参考。
